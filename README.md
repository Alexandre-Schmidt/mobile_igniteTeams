## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Expo
- Typescript
- AsyncStorage
- Phosphor React Native
- React Native Safe Area Context
- React Native Navigation
- Styled Components

## 💻 Funcionalidades:

- [x] Gerenciar turmas e times
- [x] Adicionar ou remover pessoas de times em turmas existentes
- [x] Listagem de turmas e times
- [x] Adicionar ou remover pessoas do time 
- [x] interface intuitiva e fácil de navegar

## 🚀 Instalação e uso

```bash
# Clone o repositório
git clone https://github.com/Alexandre-Schmidt/mobile_igniteTeams

# Acesse a pasta do projeto
cd mobile_igniteTeams

# Instale as dependências
npm install

# Execute a aplicação
npx expo start

# O servidor inciará na porta:3000 - acesse http://localhost:3000
```
